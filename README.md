<p>Durante o desenvolvimento de uma aplicação para encontrar perfis de pessoas no github feito no bootcamp da LOCALIZA - Digital Innovation One, criei o Git Search. </p>
<p>O Git Search é uma ferramenta para encontrar perfis de usuários e seus respectivos repositórios no github. Com base no conteúdo das aulas foram explorados mais a fundo temas como styled-components e acesso a API's, proporcionando o aprimoramento e o desenvolvimento de melhorias pra solução.</p>

<ul>
<li>Faça um fork desse repositório;</li>
